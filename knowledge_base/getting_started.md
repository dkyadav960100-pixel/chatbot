# Getting Started Guide

## Welcome to TechCorp!

This guide will help you get up and running with TechCorp products in just a few minutes.

## Step 1: Create Your Account

### Sign Up
1. Visit [techcorp.example.com/signup](https://techcorp.example.com/signup)
2. Enter your email address
3. Create a strong password (minimum 12 characters)
4. Verify your email by clicking the link we send
5. Complete your profile information

### Account Verification
For enhanced security, we verify new accounts:
- Email verification (required)
- Phone verification (optional but recommended)
- Business verification (for enterprise accounts)

## Step 3: Install Desktop Applications

### System Requirements

**Windows:**
- Windows 10 or later
- 4 GB RAM minimum (8 GB recommended)
- 500 MB available disk space
- Internet connection

**macOS:**
- macOS 11 (Big Sur) or later
- 4 GB RAM minimum (8 GB recommended)
- 500 MB available disk space
- Internet connection

**Linux:**
- Ubuntu 20.04+, CentOS 8+, or Debian 10+
- 4 GB RAM minimum
- 500 MB available disk space

### Download Links
- Windows: [Download for Windows](https://techcorp.example.com/download/windows)
- macOS: [Download for Mac](https://techcorp.example.com/download/mac)
- Linux: [Download for Linux](https://techcorp.example.com/download/linux)

### Installation Steps

**Windows:**
1. Download the installer (.exe)
2. Run the installer as Administrator
3. Follow the installation wizard
4. Launch from Start Menu

**macOS:**
1. Download the installer (.dmg)
2. Open the DMG file
3. Drag to Applications folder
4. Open from Applications

**Linux:**
```bash
# Ubuntu/Debian
wget https://techcorp.example.com/download/techcorp.deb
sudo dpkg -i techcorp.deb

# CentOS/RHEL
wget https://techcorp.example.com/download/techcorp.rpm
sudo rpm -i techcorp.rpm
```

## Common Setup Issues

### Can't Verify Email
- Check spam/junk folder
- Add noreply@techcorp.example.com to contacts
- Request new verification email
- Contact support if issues persist

### Login Problems
- Reset password using "Forgot Password"
- Clear browser cache and cookies
- Try incognito/private browsing
- Check for account lockout (5 failed attempts)

### Sync Not Working
- Check internet connection
- Verify account is active
- Restart the desktop app
- Check firewall settings
- Ensure adequate disk space

### Mobile App Issues
- Update to latest version
- Clear app cache
- Reinstall if necessary
- Check device compatibility
