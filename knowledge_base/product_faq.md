# Product FAQ - TechCorp Solutions

## General Questions

### What products does TechCorp offer?
TechCorp offers three main product lines:
1. **CloudSync Pro** - Enterprise cloud storage and collaboration platform
2. **SecureAuth** - Multi-factor authentication and identity management
3. **DataFlow Analytics** - Business intelligence and data visualization tools

### What platforms are supported?
All our products support:
- Web browsers (Chrome, Firefox, Safari, Edge)
- Windows 10/11
- macOS 11+
- iOS 14+
- Android 10+
- Linux (Ubuntu, CentOS, Debian)

### Is there a free trial?
Yes! All products offer a 14-day free trial with full functionality. No credit card required. Enterprise customers can request extended 30-day trials.

## CloudSync Pro

### What is the storage limit?
- **Basic Plan**: 100 GB per user
- **Professional Plan**: 1 TB per user
- **Enterprise Plan**: Unlimited storage

### Can I share files externally?
Yes, you can share files with external users via:
- Password-protected links
- Email invitations
- Guest access (read-only)
- Time-limited links that expire automatically

### What file types are supported?
CloudSync Pro supports all file types. Preview is available for:
- Documents (PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX)
- Images (JPG, PNG, GIF, SVG, WEBP)
- Videos (MP4, MOV, AVI, MKV)
- Code files (with syntax highlighting)

### Is there version history?
Yes, version history is available:
- Basic: 30 days
- Professional: 180 days
- Enterprise: Unlimited

## SecureAuth

### What authentication methods are supported?
SecureAuth supports multiple authentication methods:
- SMS and Voice OTP
- Email verification
- Authenticator apps (Google, Microsoft, Authy)
- Hardware tokens (YubiKey, RSA)
- Biometric authentication
- Push notifications

### Can I integrate with existing systems?
Yes, SecureAuth integrates with:
- Active Directory / LDAP
- SAML 2.0 providers
- OAuth 2.0 / OpenID Connect
- Custom REST APIs
- Popular SaaS apps (Salesforce, Office 365, Google Workspace)

### What is the uptime guarantee?
SecureAuth offers:
- 99.9% uptime SLA for Professional plans
- 99.99% uptime SLA for Enterprise plans
- Automatic failover to backup systems

## DataFlow Analytics

### What data sources can I connect?
DataFlow Analytics connects to:
- Databases: MySQL, PostgreSQL, SQL Server, Oracle, MongoDB
- Cloud services: AWS, Azure, Google Cloud
- SaaS platforms: Salesforce, HubSpot, Shopify
- Files: CSV, Excel, JSON, XML
- APIs: REST and GraphQL

### Can I create custom dashboards?
Yes! Features include:
- Drag-and-drop dashboard builder
- 50+ chart types and visualizations
- Custom themes and branding
- Interactive filters and drill-downs
- Scheduled report generation
- PDF and PowerPoint export

### Is there real-time data?
Real-time data is available on Professional and Enterprise plans:
- Streaming data connections
- Auto-refresh dashboards (configurable intervals)
- Alert notifications for threshold breaches
- Live collaboration on shared dashboards
